## 命名規則

```
表情_舌_構図.png
```

| 軸 | 例 | 備考 |
|---|---|---|
| 表情 | 穏やか / 笑顔 / 本音 / 建前 | おばあちゃんの感情・役割 |

## 却下基準（再掲）

- 舌が異常に長い（舌超長）
- 昭和おばあちゃん感が薄い
- 解像度・トリミングがスタンプ向きでない
